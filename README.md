# ace-of-slops

Проверяет русский текст на типичные ИИ-штампы и переписывает их так, чтобы текст звучал живее — без потери смысла, цифр и фактов.

Работает в [Cursor](https://cursor.com): вы открываете чат, указываете файл и просите агента прогнать текст по фреймворку.

**Репозиторий:** [github.com/jet-marketing-pr-ai-utilities/ace-of-slops](https://github.com/jet-marketing-pr-ai-utilities/ace-of-slops)
---

## Откуда это

**Оригинал (английский):** [patina](https://github.com/devswha/patina) — open-source фреймворк для поиска и правки ИИ-паттернов в тексте. Его идеи (каталог паттернов, scoring, skill для агента) легли в основу этой адаптации.

**ace-of-slops** — русская версия: ~120 паттернов под русский язык (канцелярит, «является», кальки с английского, шаблонные выводы и т.д.). Каталог собран из открытых источников — см. [SOURCES.md](SOURCES.md).

Открытый репозиторий на GitHub: [jet-marketing-pr-ai-utilities/ace-of-slops](https://github.com/jet-marketing-pr-ai-utilities/ace-of-slops). Можно клонировать, копировать в свой проект и дорабатывать под себя.

## Быстрый старт

### Шаг 1. Добавьте фреймворк в проект

Склонируйте репозиторий с GitHub или скачайте ZIP и распакуйте в корень вашего проекта в Cursor.

```bash
git clone https://github.com/jet-marketing-pr-ai-utilities/ace-of-slops.git
```

В проекте должны быть:

```text
ваш-проект/
├── ace-of-slops/                      ← склонированный репозиторий
└── .cursor/skills/ace-of-slops-ru/    ← skill для агента (скопируйте из репо)
```
> Если папки `.cursor/skills/` в корне проекта нет — скопируйте её из репозитория в корень вашего workspace.

### Шаг 2. Откройте чат в Cursor

`Ctrl+L` (или `Cmd+L` на Mac) — откроется чат с агентом.

### Шаг 3. Укажите файл и фреймворк

В поле ввода:

1. **Тегните файл**, который хотите проверить — через `@` и имя файла, например `@article.md`
2. **Тегните фреймворк** — `@ace-of-slops-ru` (skill) или папку `@ace-of-slops/`
3. **Напишите, что сделать** — одной фразой, по-русски

**Примеры промптов (можно копировать):**

```text
@article.md @ace-of-slops-ru
Прогони текст по фреймворку и перепиши ИИ-штампы. Смысл и цифры не менять.
```

```text
@draft.md @ace-of-slops-ru
Только аудит — найди ИИ-паттерны, текст не переписывай.
```

```text
@press-release.md @ace-of-slops-ru
Проверь текст на ИИ-стиль, перепиши в деловом тоне для СМИ.
```

```text
@report.md @ace-of-slops-ru
Дай оценку от 0 до 100, насколько текст похож на нейросеть.
```

Агент прочитает каталог паттернов, найдёт штампы в вашем файле и вернёт результат: список проблем, исправленный текст или оценку — в зависимости от промпта.

## Что можно попросить

| Фраза в промпте | Что получите |
|-----------------|--------------|
| «только аудит» / «не переписывай» | Список найденных ИИ-зон без правок |
| «перепиши» / «очеловечь» | Исправленный текст |
| «было / стало» | Правки по каждому паттерну |
| «оценка» / «score» | Число 0–100 и топ проблем |
| «деловой тон» / «для СМИ» | Профиль `smi` — без хайпа, с фактурой |
| «перезапиши файл» / `--in-place` | Результат сразу в исходный файл |

Slash-команда (если skill подключён): `/ace-of-slops-ru` и дальше текст или путь к файлу.

## Для продвинутых

<details>
<summary>Структура репозитория</summary>

```text
ace-of-slops/
├── README.md
├── SOURCES.md                 # откуда взяты паттерны
├── .ace-of-slops.default.yaml # настройки по умолчанию
├── .cursor/skills/ace-of-slops-ru/SKILL.md
├── patterns/ru-*.md           # 7 паков паттернов
├── lexicon/ai-ru.md           # словарь маркеров
├── profiles/                  # default, smi
└── core/                      # voice, scoring
```
Свои правила — в `custom/patterns/`, свой профиль — в `custom/profiles/`.
Настройки проекта — файл `.ace-of-slops.yaml` в корне workspace.

</details>

<details>
<summary>Опции командной строки в промпте</summary>

| Опция | Описание |
|-------|----------|
| `--audit` | только аудит |
| `--diff` | было / стало по паттернам |
| `--score` | оценка 0–100 |
| `--tone casual` / `professional` | разговорный или деловой тон |
| `--profile smi` | профиль для B2B-статей и СМИ |
| `--ouroboros` | несколько проходов до низкого score |
| `--in-place` | записать результат в файл |

</details>
