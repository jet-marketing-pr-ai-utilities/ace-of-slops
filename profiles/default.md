# Профиль: default

## Регистр

Деловой, но не канцелярский. Предпочитай активные конструкции и конкретику.

## Приоритеты правки

1. Убрать штампы и хайп
2. Сохранить числа, имена, термины, цитаты
3. Разнообразить ритм
4. Добавить голос (см. `core/voice.md`)

## pattern-overrides

Нет — все паттерны на стандартной силе.

## voice-overrides

Нет.

## Запреты

- Не добавлять факты, которых не было в оригинале
- Не смягчать жёсткие оценки без явной причины
- Не переводить термины, если они уже устоялись в тексте
