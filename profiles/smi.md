# Профиль: smi

Для материалов в стиле экспертных B2B-статей: руководители, ИТ-директоры, отраслевые специалисты.

## Регистр

Аналитический, практичный, без хайпа и рекламных превосходных степеней. Позиция — опыт внедрения, не «революция».

## Приоритеты

1. Конкретика: проценты, сроки, сценарии, trade-off
2. Фактура в каждом ключевом тезисе
3. Прогнозы отделены от фактов («по оценкам», «в отдельных кейсах»)
4. Риски и ограничения названы явно

## Усилить (amplify)

- ru-content#1 — пустые обещания без условий
- ru-style#1 — рекламные превосходные степени
- ru-filler#2 — «инновационный», «трансформационный»

## Приглушить (reduce)

- ru-communication#3 — разговорные маркеры (не превращать аналитику в блог)

## Запреты

- «лучший», «уникальный», «революционный» без доказательной базы
- «цифровая трансформация нового уровня», «максимально эффективный»
- Кейсы уровня «ИИ улучшает эффективность» без операционной сцены
- Обещание ROI без условий
